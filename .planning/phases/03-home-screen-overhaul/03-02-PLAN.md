Build a flat list interleaving `DateGroupHeader` and `TransactionCard` items:
```dart
// Group transactions by date
final grouped = groupTransactionsByDate(context, transactions);
final flatItems = <Widget>[];
for (final entry in grouped.entries) {
  // Compute daily net: sum income amounts - sum expense amounts
  final dailyNet = entry.value.fold<int>(0, (sum, tx) {
    if (tx.type == 'income') return sum + tx.amount;
    if (tx.type == 'expense') return sum - tx.amount;
    return sum; // transfers are net zero
  });
  flatItems.add(DateGroupHeader(dateLabel: entry.key, dailyNet: dailyNet));
  for (final tx in entry.value) {
    flatItems.add(_buildTransactionItem(tx, ...));
  }
}
```

Return a `SliverList.builder` with `itemCount: flatItems.length` and `itemBuilder` returning the pre-built widget at each index.
