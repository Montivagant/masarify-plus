4. Add the `_silentRestore()` method after `_checkAutoLock()`:
   ```dart
   void _silentRestore() {
     // Throttle: at most once per hour to avoid Google Play rate limits.
     final now = DateTime.now();
     if (_lastRestoreAt != null &&
         now.difference(_lastRestoreAt!).inMinutes < 60) {
       return;
     }
     _lastRestoreAt = now;
     final service = ref.read(subscriptionServiceProvider);
     // Fire and forget — errors are silently ignored.
     service.restorePurchases().catchError((_) {});
   }
   ```

This throttles the restore call to at most once per hour (using an in-memory timestamp, so it resets on app kill). The `catchError` ensures no unhandled exceptions.
</action>
